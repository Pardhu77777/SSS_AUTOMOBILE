# SSS Automobile Limited Website

Modern, production-ready marketing site for SSS Automobile Limited (auto maintenance & repair), rebuilt with Vite + React.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Routes: Home, Services, About, Contact / Booking.
- Booking form has client-side validation.
- SEO basics included in `index.html` (title, description, Open Graph defaults).
- Visual system uses layered sections, glassmorphism, gradients, and subtle parallax effects.
